# Customer_churn_analysis
This project focuses on analyzing customer churn behavior using a telecom dataset. The goal is to identify patterns, trends, and key factors that influence whether a customer leaves the service (churns) .The dataset contains detailed information about customers, account details, services subscribed, and billing information. 
The dataset includes a mix of categorical and numerical variables such as gender, senior citizen status, tenure, contract type, internet service, payment method, monthly charges, and total charges. The target variable, Churn, indicates whether a customer has discontinued the service. By performing exploratory data analysis (EDA), the project identifies trends and relationships between different features and customer churn behavior
This project utilizes tools such as Python (Pandas, NumPy, Matplotlib, Seaborn) and Power BI for data visualization. Overall, it provides valuable insights into customer behavior and serves as a foundation for building predictive models to proactively identify customers at risk of churning.
